# 개인 연습 공간입니다.

## Streamlit + CSS 사용해 본 미니 프로젝트입니다.

### 🔗 https://toeic-voca-app.streamlit.app

#### TOEIC VOCA Study App

##### 프로젝트 목적
- TOEIC 단어를 Day별로 학습하고 퀴즈로 복습하는 웹앱

##### 사용 기술
- Python
- Pandas
- Streamlit
- Custom CSS

##### 핵심 기능
- Day Study 학습 모드
- Random Word 복습
- Quiz 시스템
- Wrong Answer 노트

##### 기술적 구현 포인트
- Streamlit session_state 기반 퀴즈 상태 관리
- Pandas 기반 단어 데이터 처리
- CSS 커스터마이징으로 UI 개선
